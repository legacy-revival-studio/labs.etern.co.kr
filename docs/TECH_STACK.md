# TECH_STACK

## Why Astro

Legacy Revival Lab is primarily a content-driven showcase site.

Requirements:

- static pages
- fast load
- minimal JavaScript
- GitHub Pages deployment
- easy maintenance
- easy AI collaboration

Astro fits these requirements best.

## Architecture Principle

Legacy Revival Lab site is separate from product repositories.

- `labs.etern.co.kr` → Astro showcase
- `claytube` → separate product repo
