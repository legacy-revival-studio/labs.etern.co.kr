# TECH_STACK

## Why Astro

ETERN Labs is primarily a content-driven showcase site.

Requirements:

- static pages
- fast load
- minimal JavaScript
- GitHub Pages deployment
- easy maintenance
- easy AI collaboration

Astro fits these requirements best.

## Architecture Principle

ETERN Labs site is separate from product repositories.

Example:

- `etern.co.kr/labs` → Astro showcase
- `eternops/claytube` → separate product repo

ClayTube is linked via:
