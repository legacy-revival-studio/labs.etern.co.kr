# ETERN Labs

**Small digital products, built to last.**

ETERN Labs는 Legacy Revival Studio가 개발하는 작은 디지털 제품들을 소개하고, 제품 의뢰를 받기 위한 showcase 사이트입니다.

This repository is built with **Astro**.

---

## Tech

- Astro
- TypeScript
- GitHub Pages
- Static-first deployment

---

## Development

Install:

```bash
npm install
```

Run:

```bash
npm run dev
```

Build:

```bash
npm run build
```

Deploy:

- GitHub Pages via GitHub Actions
- Custom domain: `labs.etern.co.kr`
- The build output is published from `dist/` to the `gh-pages` branch
